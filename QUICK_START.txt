🚗 TOOSILA - QUICK START GUIDE
═══════════════════════════════════════════════════════════

⚡ GET STARTED IN 3 STEPS:

STEP 1: Setup Database (First time only)
────────────────────────────────────────
Open Command Prompt/Terminal:
  cd C:\Users\a2z\toosila-project\server
  node scripts/setup-database.js

✅ You should see: "Database schema setup completed successfully!"

STEP 2: Start Backend Server
────────────────────────────────────────
Keep terminal open:
  npm run dev

✅ You should see: "Server running on http://localhost:5000"

STEP 3: Start Frontend (New Terminal)
────────────────────────────────────────
Open NEW Command Prompt/Terminal:
  cd C:\Users\a2z\toosila-project\client
  npm start

✅ Browser opens automatically at http://localhost:3000

🐛 PROBLEMS?
═══════════════════════════════════════════════════════════

❌ Database Error?
   → Run: node scripts/setup-database.js

❌ Port 5000 in use?
   → Change PORT in server/.env

❌ CORS Error?
   → Check CORS_ORIGIN=http://localhost:3000 in server/.env

❌ Blank Page?
   → Check browser console (F12)
   → Run: npm install
