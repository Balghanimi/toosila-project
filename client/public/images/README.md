# QR Code Image

## ملاحظة مهمة

يرجى إضافة صورة QR Code للتواصل عبر تيليجرام باسم:
`telegram-qr.png`

### المواصفات المطلوبة:
- الاسم: `telegram-qr.png`
- الحجم الموصى به: 300x300 بكسل أو أكبر
- التنسيق: PNG (شفاف أو خلفية بيضاء)

### كيفية إضافة الصورة:
1. احفظ صورة الـ QR Code الخاصة بك
2. سمها `telegram-qr.png`
3. ضعها في هذا المجلد: `client/public/images/`

### ملاحظة:
إذا لم تكن الصورة موجودة، سيظهر رمز تيليجرام بديل تلقائياً.

---

الصورة المقدمة في الطلب:
- يجب حفظ صورة الـ QR Code المرفقة
- تحويلها إلى PNG إذا لزم الأمر
- وضعها في هذا المجلد
